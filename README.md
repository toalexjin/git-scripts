# Git Custom Scripts #

This project offers some git custom scripts.

### Requirements ###

* `asciidoc` is required to build man pages [(http://asciidoc.org)](http://asciidoc.org)

### Install ###

* make clean
* make build
* sudo make install

### Uninstall ###

* sudo make uninstall

### Usage Examples ###

* git sync master
* git help sync
* git pushself --all
* git help pushself
